# 📖 Project Overview

Large Language Models (LLMs) have demonstrated remarkable capabilities in generating human-like responses. However, they often suffer from **hallucination**, producing information that appears convincing but is factually incorrect.

To address this issue, this research implements **Retrieval-Augmented Generation (RAG)**, a method that combines information retrieval with language generation. By retrieving relevant external knowledge before generating responses, the model is expected to provide more accurate, contextual, and reliable answers.

---

# 🎯 Research Objectives

This study aims to:

* Evaluate the effectiveness of RAG in improving Indonesian LLM performance.
* Measure improvements in:

  * Accuracy
  * BLEU Score
  * ROUGE Score
* Compare RAG-based models against baseline models without retrieval mechanisms.
* Analyze performance across multiple knowledge domains.

---

# ❓ Research Questions

1. Can Retrieval-Augmented Generation improve Indonesian LLM performance across multiple domains?
2. How significant is the improvement in Accuracy, BLEU, and ROUGE metrics?
3. How does the RAG model compare with a conventional baseline model?
